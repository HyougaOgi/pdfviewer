# pdfViewer

Small terminal PDF viewer for Ghostty/Kitty.

## Requirements

- `pdftoppm`
- `pdfinfo`
- `ghostty` or `kitty`

## Usage

```bash
chmod +x ./pdfviewer.sh
./pdfviewer.sh /path/to/file.pdf
```

## Keys

- `n`: next page
- `p`: previous page
- `g`: jump to page
- `r`: rerender current page
- `q`: quit
